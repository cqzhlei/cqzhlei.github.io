集群实现免密码登录

准备工作

  • 保证每台节点中的用户名相同（例如都为spark）。如果没有，则创建。

    不要直接使用 root 用户。请为集群单独创建用户。

将用户添加至 sudoer 列表

  1. 切换到 root 用户（su -）。

  2. 使用 visudo 命令，添加 spark 用户到 sudoer 列表。

  3. 保存退出。

    远程拷贝id_rsa.pub给所有主机

  4. 把刚刚生成的id_rsa.pub远程拷贝给所有主机,我们这里另外一台机器名spark01
